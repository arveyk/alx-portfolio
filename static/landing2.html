IoT Description

This IoT(Internet of Things) project involves the automation of monitoring and control of a few essential parameters within a greenhouse like temperature, CO2 level and humidity.
The interface can be accessed via a browser with access to the internet. Sensosors within the green house collect data and the data is stored in a database and the application accesses the database and presents it in a user friendly manner.

About Section

What prompted me to think of working on an IoT system was the desire to design a system that simplifies greenhouse crop prduction by incorporating a system that aids monitoring and control of the greenhouse environment. This would better the yield quality and quantity when conditions are set right.
For example CO2 is used by plants in food production

Also, temperatures within the greenhouse should not be too high or plants will rapidly lose moisture and wilt and it ought not to be too low since it would stall or slow down photosynthesis.

Humidity also affects plants' uptake of water and thus affects nutrients uptake. These are just a few basic parameters that majory affect crop yield and quality besides amount of sunlight and quality of growing medium.
